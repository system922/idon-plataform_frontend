# IDON — Alianzas Estratégicas y Canales de Distribución

## 1. CONTADORES Y ASESORES FINANCIEROS

### Por qué son el canal más poderoso:
- Sus clientes les preguntan "¿qué sistema usas?" — su recomendación tiene peso
- Un contador puede traerte 5–20 clientes de un solo acuerdo
- Ellos validan la facturación electrónica — IDON la incluye, eso los convence

### Cómo acercarse:
1. Identificar 10–20 contadores / estudios contables locales
2. Ofrecer: demo gratuita + comisión del 15–20% por cada cliente referido
3. Darles acceso a cuenta de demostración para que muestren a sus clientes
4. Material: PDF de "IDON para contadores" explicando integración SRI

### Mensaje de acercamiento (LinkedIn / Email / WhatsApp):
```
Hola [Nombre],

Soy [tu nombre] de IDON, un sistema de gestión multinegocios que incluye 
facturación electrónica integrada con el SRI.

Muchos de tus clientes probablemente te preguntan qué sistema usar. 
Me gustaría mostrarte IDON y ver si encaja para recomendarlo.

A cambio, tenemos un programa de referidos donde ganas el 15% mensual 
recurrente por cada cliente que se quede.

¿Tienes 20 minutos esta semana?
```

## 2. PROVEEDORES DE HARDWARE (POS, Impresoras, Cámaras)

### Por qué funcionan:
- El cliente que compra una caja registradora también necesita software
- El proveedor hardware cierra el negocio completo si incluye el software
- Complementario, no competidor

### Acuerdo tipo:
- Proveedor hardware refiere a IDON → comisión por cliente activo
- IDON recomienda hardware compatible → comisión o descuento para el cliente final
- Co-branding: "Compatible con IDON" en el hardware

### Prioridad de contacto:
- Distribuidores de impresoras térmicas (Epson, Star Micronics)
- Proveedores de terminales POS / tablets
- Proveedores de lectores de código de barras

## 3. CÁMARAS DE COMERCIO Y GREMIOS

### Qué ofrecen:
- Base de datos de negocios locales (potenciales clientes)
- Legitimidad y credibilidad para la marca
- Acceso a eventos y ferias

### Cómo entrar:
1. Afiliarse como empresa miembro
2. Ofrecer descuento exclusivo para miembros del gremio (10–20%)
3. Participar como expositor en ferias de emprendedores
4. Dar charla gratuita: "Digitaliza tu negocio en 2025"

## 4. INFLUENCERS Y CREADORES (LATAM)

### Tipo de perfil ideal:
- Dueños de negocio que comparten su experiencia (restauranteros, emprendedores)
- Coaches de negocios y productividad
- Contadores / financieros con audiencia en Instagram o TikTok
- Seguidores: 5,000–50,000 (micro-influencers convierten mejor)

### Modelo de colaboración:
- Acceso gratuito a IDON (o comisión de afiliado)
- Pago por post/video (si tienen audiencia validada)
- No pagar por número de seguidores — medir por conversiones reales

### Brief para influencer:
```
- Prueba IDON 30 días en tu negocio real
- Comparte tu experiencia honesta (1 Reel + 3 Stories)
- Incluye tu link de afiliado único
- Tú ganas: [X]% por cada cliente que se active con tu link
```

## 6. PROGRAMA DE AFILIADOS (estructura formal)

### Niveles:
```
NIVEL 1 — Referidor
- Cualquier persona con un link único
- Gana: 10% del primer pago del cliente referido
- Sin requisitos

NIVEL 2 — Afiliado Activo  
- 5+ clientes activos referidos
- Gana: 15% recurrente mensual por cada cliente activo
- Acceso a materiales de marketing exclusivos

NIVEL 3 — Partner Oficial
- 15+ clientes activos
- Gana: 25% recurrente + soporte dedicado + co-branding
- Contrato formal, facturación mensual de comisiones
```

### Herramientas para afiliados:
- Dashboard personal con métricas de sus referidos
- Link único con seguimiento
- Materiales: banners, scripts, plantillas de WhatsApp
- Newsletter mensual con novedades del producto
